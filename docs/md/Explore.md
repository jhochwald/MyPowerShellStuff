# Explore
## SYNOPSIS
Open explorer in this directory

## SYNTAX
```powershell
Explore [[-loc] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Open the Windows Explorer in this directory
