# Get-TrImURL
## SYNOPSIS
Get a Short URL

## SYNTAX
```powershell
Get-TrImURL [-URL] <String> [<CommonParameters>]
```

## DESCRIPTION
Get a Short URL using the TR.IM Service

## PARAMETERS
