# ConvertTo-PlainText
## SYNOPSIS
Convert a secure string back to plain text

## SYNTAX
```powershell
ConvertTo-PlainText [-secure] <SecureString> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
